After a lot of thinking, we decided to discontinue this mod and move all of its contents to a new mod called Cozy Cooking. This mod will contain all the code and items from both Cozy Ingredients and Cozy Café (in the end, it just didn't make sense to have 2 separate mods lol). We've also reworked all textures to be 16x16 instead of 32x32 so it'll look much more natural and will fit Minecraft's style more.
